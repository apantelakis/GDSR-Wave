# GDSR Wave

Implementation of the geometry dash skill rating - wave google [doc](https://docs.google.com/spreadsheets/d/1M95yjFSShp_6rgBVUqjzy7dq4rfkx44Mnf-B0N5DpJs/edit?gid=0#gid=0)

Features a wave icon button in the search levels scene that opens the wave skill rating list. That's about it.
